CSS Units: px, em, and rem Demonstration

Using px (Pixels)

This section uses pixel-based values for font size and padding.

Font size: 16px

Padding: 20px

Using em

This section uses em-based values for font size and padding. The base font size here is 16px.

Font size: 1.5em (24px based on 16px parent size)

Padding: 2em (32px based on 16px parent size)

Using rem

This section uses rem-based values for font size and padding. Rem is relative to the root font size (which is 16px).

Font size: 2rem (32px)

Padding: 1.5rem (24px)
